# 使用示例

比如用户想学习spring boot的agent的写作方法

step 1: 在仓库里检索spring boot
step 2: 将检索到的文件名告诉用户
step 3: 读取减缩到的文件，根据文件内容给用户一些建议

如果在仓库找不到用户想要的agent，直接告诉用户没找到。
